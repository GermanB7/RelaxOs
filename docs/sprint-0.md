# Sprint 0 Foundation

## Scope

- Backend Spring Boot foundation with Java 21.
- PostgreSQL local through Docker Compose.
- Flyway enabled with a foundation-only migration.
- Swagger and Actuator enabled.
- React, Vite, TypeScript, Tailwind, React Router, TanStack Query, React Hook Form, and Zod installed.
- Frontend feature folders and placeholder pages for Dashboard, Scenarios, Recommendations, and Settings.

## Explicit Non-Goals

- No AI implementation.
- No score engine.
- No recommendation engine.
- No scenario CRUD.
- No domain tables.
- No native mobile app.
- No marketplace, open banking, smart home, or microservices.

## Sprint 1 Entry Points

- Define first domain migrations through Flyway.
- Add request/response contracts for the first scenario workflow.
- Introduce validation only when real requests exist.
- Add consistent API error responses once endpoints move beyond foundation status.
